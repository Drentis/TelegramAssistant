## [1.0.23] - 2026-03-29

### 🐛 Исправления

**🌤 Погода - исправление часовых поясов:**
- ✅ **Часовой пояс города** — уведомления приходят в настроенное время по часовому поясу города пользователя
- ✅ **OpenWeatherMap timezone** — используется `timezone` из API для определения локального времени
- ✅ **Кэширование часовых поясов** — часовой пояс города кэшируется для уменьшения запросов к API
- ✅ **Локальное время в прогнозе** — время дождя в уведомлениях показывается по местному времени города

**✨ Улучшения:**
- Уведомления приходят ровно в то время, которое выставил пользователь (по времени его города)
- Пример: пользователь в Москве установил 6:00 → уведомление придёт в 6:00 MSK
- Пример: пользователь в Новосибирске установил 6:00 → уведомление придёт в 6:00 NOVT
- Исправлена проблема, когда уведомления приходили в 9:00 вместо 6:00

## Формат версий

Формат: `[MAJOR.MINOR.PATCH]`

- **MAJOR** — крупные изменения, несовместимые с предыдущей версией
- **MINOR** — новые возможности, совместимые с предыдущей версией
- **PATCH** — исправления ошибок, совместимые с предыдущей версией
