# 📝 Changelog

Все изменения в проекте TelegramAssistant.

---

## [1.0.5] - 2026-03-21

### ✨ Улучшения

**📦 Установка:**
- Добавлен отдельный скрипт install.sh для установки одной командой
- Скрипт загружает deploy.sh во временную папку и запускает локально
- Интерактивный ввод теперь работает корректно

### 🐛 Исправления

**⌨️ Интерактивный ввод:**
- Все команды read используют < /dev/tty
- Токены и ID запрашиваются при установке

---

## [1.0.4] - 2026-03-21

### 🐛 Исправления

**⌨️ Интерактивный ввод при установке:**
- Исправлена проблема с пропуском ввода токенов при установке через `curl | bash`
- Все команды `read` теперь используют `< /dev/tty` для работы с pipe
- Токены и ID запрашиваются корректно

## [1.0.2] - 2026-03-21

### 🐛 Исправления

**🔄 Обновление бота:**

## [1.0.1] - 2026-03-21

### 🐛 Исправления

**🌤 Погода:**
- Исправлена проблема с отправкой уведомлений о погоде в заданное время
- Добавлено преобразование значений БД (weather_daily, weather_rain) в bool

## [1.0.0] - 2026-03-21

### ✨ Первый релиз TelegramAssistant

**🎯 Основные возможности:**
- 🛒 **Список покупок** с автоматической классификацией (Магнит/Фикспрайс/Другое)
- 📋 **Список дел** с напоминаниями и датами выполнения
- 📚 **Учёба** — отдельный список для учебных задач
- 💡 **Идеи** — быстрые заметки
- 🍳 **Рецепты** с ингредиентами и кнопкой "Добавить в корзину"
- 🌤 **Погода** — ежедневные прогнозы и уведомления о дожде

**⚙️ Настройки:**
- Настройка названий магазинов и сокращений
- Триггерные слова для команд
- Видимость кнопок меню
- Настройки погоды (город, время, уведомления)

**🔧 Админ-панель:**
- Статистика пользователей и записей
- Обновление бота через Telegram
- Команда `telegramactl` для управления сервисом

**📦 Установка:**
- Автоматическая установка на Linux (systemd)
- Установка на Windows (PowerShell)
- Публичный репозиторий на GitHub

## Формат версий

Формат: `[MAJOR.MINOR.PATCH]`

- **MAJOR** — крупные изменения, несовместимые с предыдущей версией
- **MINOR** — новые возможности, совместимые с предыдущей версией
- **PATCH** — исправления ошибок, совместимые с предыдущей версией
