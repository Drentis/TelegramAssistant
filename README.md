## ✨ Возможности

### 🛒 Список покупок
- **Автоматическая классификация** товаров по магазинам:
  - **Магнит** — продукты питания (молоко, хлеб, мясо, овощи, фрукты и т.д.)
  - **Фикспрайс** — бытовые товары (шампунь, порошок, посуда, канцелярия и т.д.)
  - **Другое** — всё остальное
- **Ручное распределение** с помощью префиксов магазинов
- **Отметка купленных товаров** (скрываются из списка)
- **Пакетное добавление** через запятую

### 📋 Список дел
- Добавление задач с **датами выполнения**
- **Напоминания** о делах (ежедневно в 9:00)
- Распознавание дат в форматах:
  - `завтра`, `послезавтра`, `сегодня`
  - `15.03`, `15.03.2024`
  - `15 марта`, `15-го марта`
  - `на 15.03`, `на завтра`

### 📚 Учёба
- Отдельный список для учебных задач
- Без дат и напоминаний
- Быстрое добавление и удаление

### 💡 Идеи
- Быстрые заметки для важных идей
- Мгновенное сохранение
- Просмотр всех идей в одном месте

### 🍳 Рецепты
- Сохранение рецептов с названиями и описаниями
- Добавление ингредиентов по одному
- **Кнопка "Добавить в корзину"** — автоматическое добавление всех ингредиентов в список покупок
- Просмотр списка рецептов

### 🌤 Погода
- **Ежедневные прогнозы** в заданное время
- **Уведомления о дожде** (возьмите зонт!)
- Настройка города и времени отправки
- Использование API OpenWeatherMap

### ⚙️ Настройки
- **Название магазинов** (например, "Перекрёсток" вместо "Магнит")
- **Сокращения** для быстрого ввода (м, ф, д)
- **Описания** категорий
- **Триггерные слова** для команд (купить, сделать, учёба, идея, рецепт)
- **Видимость кнопок** меню (можно отключить ненужные разделы)
- **Настройки погоды** (город, время, уведомления)

## 🚀 Запуск

### 🖥️ На VPS (рекомендуется)

После установки через `install.sh` бот запускается автоматически как systemd сервис:

```bash
# Проверка статуса
telegramactl status

# Просмотр логов
telegramactl logs

# Перезапуск
telegramactl restart
```

Бот будет:
- ✅ Запускаться автоматически при загрузке сервера
- ✅ Перезапускаться при ошибках
- ✅ Работать 24/7 в фоновом режиме

### Локальный запуск

### Обычный запуск
```bash
python main.py
```

### Запуск в фоновом режиме (Linux/Mac)
```bash
nohup python main.py &
```

### Запуск через systemd (вручную)

Создайте файл `/etc/systemd/system/telegramassistant.service`:

```ini
[Unit]
Description=TelegramAssistant Bot
After=network.target

[Service]
Type=simple
User=telegramassistant
WorkingDirectory=/opt/telegramassistant
ExecStart=/opt/telegramassistant/venv/bin/python /opt/telegramassistant/main.py
Restart=always

[Install]
WantedBy=multi-user.target
```

Запустите:
```bash
sudo systemctl daemon-reload
sudo systemctl enable telegramassistant
sudo systemctl start telegramassistant
```

## 📖 Использование

### Первые шаги
1. Запустите бота командой `/start`
2. Бот покажет приветственное сообщение с доступными командами
3. Выберите раздел из меню или используйте текстовые команды

### 🛒 Добавление покупок

**Автоматическая классификация:**
```
купить молоко, яйца, хлеб
→ молоко → Магнит
→ яйца → Магнит
→ хлеб → Магнит

купить шампунь, губка
→ шампунь → Фикспрайс
→ губка → Фикспрайс

купить ручка, игрушка
→ ручка → Фикспрайс
→ игрушка → Фикспрайс

купить что-то необычное
→ что-то необычное → Другое
```

**Ручное распределение по магазинам:**
```
м купить батон          → Магнит
ф купить тряпка         → Фикспрайс
д купить подарок        → Другое

Магнит купить сыр       → Магнит
Фикспрайс купить мыло   → Фикспрайс
```

**Пакетное добавление:**
```
купить молоко, хлеб, сыр, колбаса
→ Все товары добавятся в соответствующие категории
```

### 📋 Добавление дел

**Без даты:**
```
сделать уборку
→ Задача добавлена без срока
```

**С датой:**
```
сделать звонок завтра
→ Напоминание придёт завтра

сдать проект 25.03
→ Напоминание придёт 24.03

прочитать книгу 15 марта
→ Напоминание придёт 14.03

написать отчёт на 30.03.2024
→ Напоминание придёт 29.03.2024
```

### 📚 Добавление учебных задач
```
учёба выучить 50 слов
учёба прочитать главу 5
учёба решить 10 задач
```

### 💡 Добавление идей
```
идея записать сон
идея для стартапа
идея подарка на день рождения
```

### 🍳 Добавление рецептов

**Процесс добавления:**
```
1. рецeпт Борщ
   → Бот предложит добавить ингредиенты

2. свёкла 3шт
   → Добавлено

3. картофель 4шт
   → Добавлено

4. мясо 500г
   → Добавлено

5. готово (или /done)
   → Бот предложит добавить описание

6. Варить 2 часа, добавить капусту
   → Рецепт сохранён!
```

**Использование рецепта:**
- Откройте раздел "Рецепты"
- Выберите рецепт
- Нажмите "Добавить в корзину"
- Все ингредиенты добавятся в список покупок

### 🌤 Настройка погоды

1. Откройте **⚙️ Настройки**
2. Выберите **🌤 Погода**
3. Настройте:
   - **Город** (например, Москва)
   - **Время отправки** (например, 07:00)
   - **Утренний прогноз** (вкл/выкл)
   - **Уведомление о дожде** (вкл/выкл)

## ⏰ Планировщики

### reminder_scheduler
- **Время запуска:** Ежедневно в 9:00
- **Описание:** Проверяет задачи со сроком на завтра и отправляет напоминания
- **Частота проверки:** Каждую минуту

### weather_scheduler
- **Время запуска:** В настроенное пользователем время
- **Описание:** Отправляет прогноз погоды и уведомления о дожде
- **Частота проверки:** Каждую минуту

## 🐛 Устранение неполадок

### Бот не запускается
1. Проверьте токен в `.env`
2. Убедитесь, что все зависимости установлены: `pip install -r requirements.txt`
3. Проверьте логи на наличие ошибок

### Не работает погода
1. Проверьте API ключ в `.env`
2. Убедитесь, что город установлен в настройках
3. Проверьте, включены ли уведомления в настройках погоды

### База данных не создаётся
1. Убедитесь, что у бота есть права на запись в директорию
2. Проверьте, не блокирует ли антивирус файл `.db`

### Напоминания не приходят
1. Проверьте, что бот запущен постоянно
2. Убедитесь, что задачи имеют дату выполнения
3. Напоминания отправляются только о задачах на завтра
